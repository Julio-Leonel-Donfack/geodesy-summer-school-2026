# Geodesy Summer School 2026 – Group 1

## Sea Level Rise & Coastal Flooding in the Gulf of Guinea

This repository contains the code, data and figures produced during the Geodesy for a Resilient Africa Summer School (AIMS Cameroon, June–July 2026).

### Project overview
- Multi‑proxy assessment combining satellite altimetry, InSAR and GRACE
- Case study: Douala (Cameroon) and the Gulf of Guinea
- Python notebooks for SSH time‑series, subsidence mapping, and vulnerability analysis
